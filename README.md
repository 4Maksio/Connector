# Connector (PL) [Praca w toku]

Aplikacja MVVM tworzona w .Net 8.0 przy uzyciu frameworka Avalonia UI

## Prototyp 1

Oznaczenie wersji: **0.0.1**  
Pokrycie wymagań: **88,89%**  
Główne funkcjonalności

- Ekran powitalny
- Dashboard
- Tworzenie i usuwanie obiektów (ochrona przed przypadkowym usunięciem)

Instalator **win-x64**: [ConnectorSetup(0.0.1)](https://github.com/4Maksio/Connector/blob/main/win-x64/ConnectorSetup(0.0.1).exe) (kliknij ikonę pobrania po prawej stronie ekranu)
